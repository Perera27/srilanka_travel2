LANKA EXPLORER - RUN STEPS

1) Open terminal in this project folder
2) Create virtual environment
   Windows:
   py -m venv venv
   venv\Scripts\activate

   Mac/Linux:
   python3 -m venv venv
   source venv/bin/activate

3) Install packages
   pip install -r requirements.txt

4) Apply migrations
   python manage.py migrate

5) Seed sample data and admin account
   python manage.py seed_data

6) Start server
   python manage.py runserver

7) Open in browser
   http://127.0.0.1:8000/

Admin login:
username: admin
password: admin1234

If you edit models later:
python manage.py makemigrations
python manage.py migrate

If static files look broken:
python manage.py collectstatic

If you get missing package errors, make sure the virtual environment is activated before running any command.
