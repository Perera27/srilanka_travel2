About Lanka Explorer

A final year project by Bulathsinghalage Dinuk Nemitha Perera

🎯 Project Aim

Lanka Explorer is a web-based travel information system developed to help both domestic and international travellers discover Sri Lanka's incredible destinations — from popular beaches to hidden cultural gems. The system provides personalised recommendations based on user preferences, location, and interests.

🛠️ Technology Stack
  • Backend: Django (Python)
  • Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
  • Database: SQLite
  • Maps: Leaflet.js + OpenStreetMap
  • Tools: GitHub, Figma, Trello
